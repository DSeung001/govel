# MVP Non-goals

Govel MVP는 아래 루프만 완성하는 것이 목표입니다.

World → Proposal → AI Review → Approve → Markdown Update → Publish

이 범위를 벗어나는 기능은 추가하지 않습니다.

## Do Not Add

- Do not add a database.
- Do not add login.
- Do not add Discussions integration.
- Do not add giscus.
- Do not add Notion import.
- Do not add Obsidian import.
- Do not add AI model selection UI.
- Do not add custom API key input UI.
- Do not add a complex dashboard.
- Do not add graph visualization.
- Do not add novel generation.
